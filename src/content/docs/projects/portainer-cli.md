A **command-line interface** for managing Portainer container environments from the terminal.

**GitHub:** [github.com/cbridges1/portainer-cli](https://github.com/cbridges1/portainer-cli)

---

## Overview

Portainer CLI wraps the Portainer API to allow scripted and terminal-driven management of Docker environments — useful for CI/CD pipelines and automated infrastructure workflows.

## Tech Stack

**Language** — Go

**Binary** — Single compiled binary, no runtime dependencies

## Key Features

- Manage Portainer stacks and containers from the command line
- Suitable for automation and CI/CD integration
- Single compiled binary, zero runtime dependencies
