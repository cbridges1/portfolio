# React Native AdMob Native Ads

A library for rendering **AdMob Native Advanced Ads** in React Native applications using true native views.

**GitHub:** [github.com/cbridges1/react-native-admob-native-ads](https://github.com/cbridges1/react-native-admob-native-ads)

---

## Overview

Unlike banner-based ad solutions, this library renders ads using native Android view components, giving full layout control while maintaining AdMob compliance. Built as a bridge between the React Native JS layer and the native AdMob SDK.

## Tech Stack

| Layer | Details |
|---|---|
| Native | Java (Android native module) |
| Bridge | React Native JS bridge |
| Ads SDK | Google AdMob (Android) |

## Key Features

- Native view rendering for AdMob Native Advanced Ads
- Flexible ad layout via React Native components
- Compatible with EAS Build (Expo Application Services)
