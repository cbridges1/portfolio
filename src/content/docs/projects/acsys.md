## Overview

Acsys is designed to get a CMS running with minimal configuration. It connects to your existing database and exposes content through a clean API — no proprietary data formats, no lock-in.

## Tech Stack

**Backend** — Node.js with Express

**Frontend** — React admin UI built with Vite

**Databases** — SQLite, MySQL, Google Cloud Firestore

**Cloud** — First-class GCP / Firebase integration

## Key Features

- Headless REST API over any supported database
- React-based admin panel (Vite)
- Multi-database: SQLite, MySQL, Cloud Firestore
- GCP-native deployment support
- Easy self-host install
