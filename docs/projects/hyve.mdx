# Hyve

**Go-based developer infrastructure project**, with a companion CLI proof-of-concept in `hyve-cli-poc`.

**GitHub:** [github.com/cbridges1/hyve](https://github.com/cbridges1/hyve)

---

## Overview

Hyve explores developer tooling patterns in Go, focusing on infrastructure automation and service management. The `hyve-cli-poc` repo extends this with a command-line interface for interacting with the system.

## Tech Stack

**Core** — Go

**CLI** — `hyve-cli-poc` — companion command-line interface in Go
